┌──────────────────────────────────────────────────────────────────────────────┐
│ VALIDATION FLOW                                                              │
├──────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│ User Upload Image                                                          │
│         ↓                                                                    │
│ ┌─────────────────────────────────────────────────────────────┐             │
│ │ VALIDATION CHECK 1: Is Agricultural?                        │             │
│ │ • Gemini Vision analyzes image                             │             │
│ │ • Returns: field_type, detected_crops, confidence          │             │
│ └─────────────────────────────────────────────────────────────┘             │
│         ↓                                                                    │
│    Is Agricultural?                                                        │
│    ├─ NO  ❌ → Return error: "Not agricultural"              │             │
│    │         • Show detected content                         │             │
│    │         • Recommendation: Upload farm image             │             │
│    │         • STOP - FAIL VALIDATION                        │             │
│    │                                                         │             │
│    └─ YES ✅ → Continue to Check 2                           │             │
│         ↓                                                                    │
│ ┌─────────────────────────────────────────────────────────────┐             │
│ │ VALIDATION CHECK 2: Crop Type Match? (if crop specified)    │             │
│ │ • Compare detected crop to selected crop                   │             │
│ │ • Minimum confidence threshold: 50%                        │             │
│ └─────────────────────────────────────────────────────────────┘             │
│         ↓                                                                    │
│    Crop matches?                                                           │
│    ├─ NO  ❌ → Return error: "Detected rice, not wheat"     │             │
│    │         • Show identified crop                         │             │
│    │         • Recommendation: Match crop type               │             │
│    │         • STOP - FAIL VALIDATION                        │             │
│    │                                                         │             │
│    └─ YES ✅ → PASS VALIDATION                               │             │
│         ↓                                                                    │
│ ┌─────────────────────────────────────────────────────────────┐             │
│ │ PROCEED TO IMAGE ANALYSIS                                   │             │
│ │ • check-readiness                                          │             │
│ │ • grade-quality                                            │             │
│ │ • detect-damage                                            │             │
│ │ • detect-spoilage                                          │             │
│ │ • detect-phase                                             │             │
│ └─────────────────────────────────────────────────────────────┘             │
│         ↓                                                                    │
│ Return Recommendations (with validation metadata)                          │
│         ↓                                                                    │
│ Frontend Displays Results                                                  │
│                                                                              │
